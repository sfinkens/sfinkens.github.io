Introduction¶

Supported Data Format¶

Pygac reads AVHRR GAC (Global Area Coverage) and LAC (Local Area Coverage) level 1b data from NOAA, which is described in the POD (NOAA-14 and before) and KLM (NOAA-15 and following) user guides. The data can be obtained from NOAA CLASS, where you can also find a comprehensive introduction.

Note

Pygac is currently not able to read files with the CLASS archive header included.

Supported Sensors¶

Pygac currently supports AVHRR generations 1-3 onboard NOAA (TIROS-N, NOAA-6 and onwards) and MetOp satellites.

Related Projects¶

  • pygac-fdr: Generate a fundamental data record of AVHRR GAC data using Pygac.

  • level1c4pps: Prepare AVHRR GAC data for NWCSAF/PPS using Pygac.
